기존의 정적인 비디오 생성의 함정에서 벗어나(Escaping the static video trap), 다차원적이고 동적인 서사를 완벽하게 통제하는 **N-Anchor (Narrative-Anchored Training-Free Generation)**의 A to Z입니다.

# Title: Beyond the Prompt Trap: A Training-Free Paradigm for Narrative-to-Video Generation via Global Context Anchoring

## 1. Idea (핵심 아이디어)

본 연구의 핵심 아이디어는 비디오 생성 모델의 입력 패러다임을 $P \rightarrow V$ (Prompt-to-Video)에서 $N \rightarrow V$ (Narrative-to-Video)로 전환하는 것입니다.
날것의 서사 텍스트(Raw Narrative)가 주어졌을 때, 대형 언어 모델(LLM)을 통해 전체 서사를 관통하는 **'글로벌 앵커(등장인물, 배경)'**를 추출합니다. 이후, 시각적 묘사가 생략된 서사 문장들을 이 앵커에 종속시켜 '동적 프롬프트'로 번역하고, 사전 학습된 비디오 확산 모델(Diffusion Model)의 **Cross-Attention 계층에 앵커의 특징(Feature)을 직접 주입(Injection)**합니다. 이를 통해 추가적인 모델 파라미터 업데이트(Training-free) 없이도 멀티 샷 환경에서 완벽한 서사적/시각적 일관성을 유지합니다.

## 2. Motivation (연구 동기)

* **The Illusion of T2MSV:** 현재 학계의 Text-to-Multi-Shot Video 모델들은 사실상 기계적으로 잘 가공된 시각적 프롬프트(Explicit Prompt)에 의존하고 있습니다. 이는 진정한 텍스트 이해가 아닙니다.
* **Domain Gap in Modality:** 실제 소설이나 대본(Narrative)은 시각적 정보가 희소(Sparse)하며, 대화와 은유 위주로 구성됩니다. 기존 모델에 이 날것의 텍스트를 넣으면, 모델은 빈 공간을 임의로 채우려다 심각한 환각(Hallucination)을 일으키고, 샷이 전환될 때마다 엔티티(인물/배경)가 증발하거나 변형됩니다.
* **Cost of Consistency:** 일관성을 유지하기 위해 로라(LoRA)를 깎거나 어댑터(Adapter)를 추가 학습하는 방식은 천문학적인 컴퓨팅 비용을 요구합니다.
* **Goal:** 따라서 텍스트의 구조적 한계를 추론 타임(Inference time)의 알고리즘만으로 극복하여, 텍스트와 비디오 사이의 도메인 갭을 무너뜨리는 것이 이 연구의 강력한 동기입니다.

## 3. Concrete Algorithm (구체적인 알고리즘 및 수식)

### 3.1 Problem Formulation

기존 모델은 명시적 프롬프트 시퀀스 $P = {p_1, p_2, \dots, p_T}$에 의존하여 샷 $v_t$를 생성합니다.


$$V = \prod_{t=1}^{T} \mathcal{G}(v_t | p_t, v_{t-1})$$


우리는 날것의 텍스트 $\mathcal{N}$으로부터 추출된 글로벌 앵커 $\mathcal{A}$와 동적 프롬프트 $P'$를 활용하는 Training-free 함수 $\mathcal{G}_{free}$를 제안합니다.


$$V = \prod_{t=1}^{T} \mathcal{G}_{free}(v_t | p'_t, \mathcal{A})$$

### 3.2 Phase 1: Global Narrative Anchoring (글로벌 서사 디코딩)

소설 텍스트 $\mathcal{N}$을 LLM 에이전트 $\Psi$에 입력하여, 서사 전체를 지배하는 정체성(Identity)을 명시적 속성으로 디코딩합니다.


$$\mathcal{A}_{entity}, \mathcal{A}_{bg} = \Psi_{extract}(\mathcal{N})$$

* $\mathcal{A} = {\mathcal{A}*{entity}, \mathcal{A}*{bg}}$는 비디오 생성 전반에 걸쳐 절대 변하지 않는 상수로 작용합니다.

### 3.3 Phase 2: Visual Hallucination Control (시각적 공백 보간)

서사 문장 $s_t \in \mathcal{N}$은 시각적 묘사가 부족할 수 있습니다. LLM 번역기 $\Psi_{translate}$는 $\mathcal{A}$를 제약 조건으로 삼아 $s_t$를 시각적 프롬프트 $p'_t$로 맵핑합니다.


$$p'_t = \Psi_{translate}(s_t | \mathcal{A}) \quad \forall t \in \{1, \dots, T\}$$


이로 인해 "그녀가 고개를 끄덕였다"라는 문장이 "금발 머리에 붉은 코트를 입은 여자가 공원 벤치에 앉아 고개를 끄덕인다"로 안전하게 확장됩니다.

### 3.4 Phase 3: Anchor-Injected Attention (Training-Free 생성)

사전 학습된 Diffusion 모델 $\epsilon_\theta(x_t, t, c)$의 디노이징 과정에서 Cross-Attention 맵을 직접 조작합니다. 텍스트 인코더 $\tau_\theta$를 통과한 특징 벡터를 활용합니다.
일반적인 Attention은 Query $Q_{x}$, Key $K_{p'_t}$, Value $V_{p'*t}$로 연산됩니다. N-Anchor는 여기에 $\mathcal{A}$의 임베딩 $K*{\mathcal{A}}, V_{\mathcal{A}}$를 결합 행렬(Concatenated Matrix)로 만들어 강제 주입합니다.

$$\tilde{K} = [K_{p'_t} \parallel K_{\mathcal{A}}]$$

$$\tilde{V} = [V_{p'_t} \parallel V_{\mathcal{A}}]$$

$$\text{Attention}_{N-Anchor} = \text{Softmax}\left(\frac{Q_{x} \tilde{K}^T}{\sqrt{d}}\right)\tilde{V}$$

더 정교한 제어를 위해 시간 스텝 $t$에 따른 주입 가중치 $\lambda$를 도입하여 듀얼 어텐션(Dual Attention) 형태로 구성할 수도 있습니다.


$$\text{Attention} = \text{Softmax}\left(\frac{Q_{x} K_{p'_t}^T}{\sqrt{d}}\right)V_{p'_t} + \lambda \cdot \text{Softmax}\left(\frac{Q_{x} K_{\mathcal{A}}^T}{\sqrt{d}}\right)V_{\mathcal{A}}$$


이 수식을 통해 네트워크는 매 프레임을 렌더링할 때마다 동적 샷의 행동($p'_t$)과 캐릭터의 본질($\mathcal{A}$)을 동시에 바라보게 됩니다.

## 4. Experiment Settings (실험 세팅)

**4.1 Datasets (Narrative Benchmark)**

* **ROCStories (Story Cloze Test):** 5문장으로 구성된 기승전결 일상 서사 1,000개 추출. (5-Shot 비디오 평가용)
* **WritingPrompts:** 극도의 시각적 모호성과 은유를 가진 Reddit 창작 단편 소설 500개.

**4.2 Baselines (비교 모델)**

* **Zero-Shot T2V:** LTX-Video, AnimateDiff (정통파 베이스라인)
* **Story-to-Video:** StoryDiffusion, VGoT (최신 경쟁 모델)

**4.3 Evaluation Metrics (평가 지표)**

* **Entity Consistency (SCENE Coherence):** DINOv2 및 CLIP-I를 사용하여 샷 $t$와 $t+1$ 사이의 메인 캐릭터 특징 벡터 코사인 유사도 측정.
* **Narrative Alignment:** 비디오를 Video-LLM(Video-LLaVA)으로 캡셔닝한 후, 원본 날것의 텍스트 $\mathcal{N}$과의 BERTScore 및 밀도 기반 교차 정렬 점수 측정.
* **Fidelity:** 기존의 FVD (Frechet Video Distance) 유지.

**4.4 Implementation Details**

* Backbone: WAN (최신 고화질 모델) 또는 AnimateDiff.
* LLM Agent: GPT-4o API 또는 LLaMA-3-8B.
* Hardware: NVIDIA A100 GPU 1대로 추론(Inference) 단계만 수행.

## 5. Full Experiments (예상 실험 결과 시나리오)

**5.1 Main Results (정량적 성능 압살)**

* 기존의 친절한 'Explicit Prompt' 세팅에서는 베이스라인(VGoT, StoryDiffusion)과 N-Anchor가 비슷한 Consistency(약 0.85 CLIP-I)를 보입니다.
* 그러나 **'Raw Narrative' 데이터셋**을 넣는 순간, 베이스라인들의 Consistency는 0.4 이하로 폭락합니다(캐릭터 증발). 반면 N-Anchor는 **0.82 이상을 방어**하며 도메인 갭을 완벽히 해소했음을 데이터로 증명합니다.
* Narrative Alignment 지표에서도 기존 모델들은 빈 공간을 마음대로 상상(Hallucination)하여 원작 텍스트와의 의미적 거리가 멀어지지만, 우리는 $\mathcal{A}$ 제약을 통해 높은 점수를 기록합니다.

**5.2 Qualitative Results (정성적 결과 시각화)**

* 그림(Figure) 구성: ROCStories의 5문장을 입력했을 때, 베이스라인들은 샷 1에서는 백인 남성이었다가 샷 3에서 흑인 여성으로 변하지만, N-Anchor는 옷의 주름과 배경의 톤까지 그대로 유지하며 자연스럽게 동작만 변하는 멀티 샷 시퀀스를 보여줍니다.

**5.3 Ablation Study (절제 연구)**

* **w/o Phase 1 (No Anchor Extraction):** $\mathcal{A}$ 없이 문장 번역만 수행한 경우, 샷마다 옷 색깔이 미세하게 바뀌는 현상 발생.
* **w/o Phase 3 (No Attention Injection):** $\mathcal{A}$를 텍스트 프롬프트에 글로만 붙여준 경우($p'_t + \mathcal{A}$), 모델이 긴 프롬프트를 무시(Catastrophic Neglect)하고 특정 샷에서 앵커 정보를 놓침. 오직 Attention Matrix 단에서 강제 주입해야만 완벽한 일관성이 보장됨을 입증.

****상세 실험*****
1. 실험 데이터셋 구성 (The Narrative Benchmark)
기존의 친절한 프롬프트 데이터셋(예: WebVid, UCF101)으로는 우리의 우수성을 증명할 수 없습니다. 서사적 텍스트를 다루는 새로운 평가 기준셋을 구축(또는 기존 NLP 데이터셋 가공)해야 합니다.

Raw Narrative Set (날것의 서사): 실제 영미권 소설, 시나리오(예: Gutenberg, MovieNet)에서 추출한 5~6문장 길이의 연속된 서사 텍스트. 시각적 묘사가 생략되거나 은유가 포함된 텍스트여야 합니다.

Explicit Prompt Set (대조군): 위 소설 텍스트를 LLM(GPT-4 등)을 이용해 기존 모델들이 좋아하는 '완벽한 시각적 프롬프트(캐릭터 외형, 배경, 조명 등을 꽉 채운 형태)'로 번역한 텍스트.

실험 목적: 기존 모델들은 Raw Narrative에서 성능이 폭락하지만, 우리의 N-Anchor 알고리즘은 성능 저하 없이 일관성을 유지한다는 점을 보여줍니다.

2. 비교 대상 베이스라인 (Baselines)
최신 오픈소스 및 멀티 샷(Multi-Shot) 비디오 생성 모델들을 도마 위에 올립니다.

Standard T2V (정통파): LTX-Video, AnimateDiff (프롬프트를 단순히 순차적으로 넣었을 때 캐릭터가 어떻게 증발하는지 보여주는 용도)

Story-to-Video (경쟁자): StoryDiffusion, VGoT 등 현재 멀티 샷 생성을 타겟팅하는 모델들. 이들에게 날것의 서사를 주었을 때 환각(Hallucination)으로 인해 서사가 붕괴됨을 증명합니다.

3. 평가 지표 (Evaluation Metrics)
단순 화질 평가(FID, FVD)는 서브로 빼고, 멀티 샷 비디오 생성 환경에서의 SCENE(서사적/시각적 일관성) 평가에 집중해야 합니다.

정량적 지표 (Quantitative):

Entity Consistency (엔티티 유지율): 첫 샷의 주인공 얼굴/옷차림이 마지막 샷까지 유지되는지 DINOv2나 CLIP-I를 통해 샷 간의 이미지 유사도를 측정합니다.

Narrative Alignment (서사 정렬도): 생성된 비디오를 Video-LLM(예: Video-LLaVA)으로 다시 텍스트로 캡셔닝한 뒤, 원래의 소설 텍스트와 의미론적으로 얼마나 일치하는지(Density-Normalized score) 측정합니다.

정성적 지표 (Qualitative / User Study):

동일한 소설 텍스트를 주고 생성된 영상들을 사람들에게 보여주며 "어느 영상이 원작 소설의 분위기와 캐릭터를 가장 잘 유지하면서 샷을 전환했는가?"를 블라인드 테스트합니다.

4. 절제 연구 (Ablation Study)
우리 알고리즘(N-Anchor)이 "왜" 작동하는지 내부 부품을 하나씩 빼보며 증명하는 단계입니다. 심사위원들이 가장 꼼꼼하게 보는 곳입니다.

w/o Global Anchor Extraction: 소설 전체에서 캐릭터/배경 앵커를 뽑지 않고, 문장별로만 LLM에 넘겼을 때의 결과 (→ 샷마다 주인공 얼굴이 바뀌는 현상 발생 증명).

w/o Attention Injection (Phase 3 제거): 앵커를 뽑기만 하고 Diffusion 모델의 Cross-Attention에 강제로 주입하지 않았을 때의 결과 (→ Training-free 제어의 필수성 증명).





*****상세수식********
핵심 타겟은 기존의 $P \rightarrow V$ (Prompt-to-Video) 패러다임을 깨부수는 **진정한 $N \rightarrow V$ (Narrative-to-Video)**이며, 이를 컴퓨팅 자원의 소모 없이 이루어내는 Training-Free 구조입니다.이 파이프라인의 가칭을 **N-Anchor (Narrative-Anchored Training-Free Generation)**라고 명명하고 수식화를 시작하겠습니다.1. 문제 정의 (Problem Formulation: P2V vs. N2V)먼저 기존 모델들의 한계를 수학적으로 정의하여 N2V의 당위성을 확보합니다.기존 P2V (Prompt-to-Video):기존 모델은 시각적 묘사가 꽉 찬 프롬프트 시퀀스 $P = {p_1, p_2, \dots, p_T}$가 주어질 때 비디오 $V$를 생성합니다.$$V = \prod_{t=1}^{T} \mathcal{G}(v_t | p_t, v_{t-1})$$여기서 $\mathcal{G}$는 비디오 생성 모델입니다. 이 방식은 날것의 서사 텍스트 $N$이 들어오면 시각적 정보의 희소성(Sparsity) 때문에 $\mathcal{G}$의 조건부 확률이 붕괴됩니다.제안하는 N2V (Narrative-to-Video):우리는 날것의 서사 텍스트 $\mathcal{N}$으로부터 전역적 일관성을 유지하는 글로벌 앵커(Global Anchor) $\mathcal{A}$를 추출하고, 이를 기반으로 시각적 공백을 채운 동적 프롬프트(Dynamic Prompts) $P'$를 도출하여 비디오를 생성합니다.$$V = \prod_{t=1}^{T} \mathcal{G}_{free}(v_t | p'_t, \mathcal{A})$$여기서 $\mathcal{G}_{free}$는 어떠한 파라미터 업데이트도 필요 없는(Training-free) 제어된 확산(Diffusion) 과정입니다.2. Training-Free N2V 알고리즘 파이프라인알고리즘은 크게 세 단계(Phase)로 나뉩니다.Phase 1: 글로벌 서사 디코딩 및 앵커 추출 (Global Narrative Anchoring)날것의 소설 텍스트 $\mathcal{N}$을 통째로 분석하여, 전체 서사를 관통하는 핵심 엔티티(인물, 물건)와 배경을 추출합니다.대형 언어 모델(LLM) $\Psi$를 사용하여 서사 속 암시적 정보를 명시적 속성으로 디코딩합니다.$$\mathcal{A}_{entity}, \mathcal{A}_{bg} = \Psi_{extract}(\mathcal{N})$$$\mathcal{A}_{entity}$: 등장인물들의 일관된 외형 묘사 (예: "흉터가 있는 전사, 붉은 망토")$\mathcal{A}_{bg}$: 주요 사건이 일어나는 시공간적 배경Phase 2: 시각적 공백 보간 및 동적 프롬프트 번역 (Visual Hallucination Control)소설은 "그가 웃었다"처럼 시각적 묘사가 비어있는 구간이 많습니다. 텍스트 $\mathcal{N}$을 샷 단위 문장 ${s_1, s_2, \dots, s_T}$로 분할한 뒤, $\mathcal{A}$를 제약 조건(Constraint)으로 삼아 시각적 프롬프트 $p'_t$로 번역합니다.$$p'_t = \Psi_{translate}(s_t | \mathcal{A}_{entity}, \mathcal{A}_{bg}) \quad \forall t \in \{1, \dots, T\}$$이 과정에서 샷이 전환되더라도 인물의 옷차림이나 배경이 앵커 $\mathcal{A}$에 종속되므로, LLM이 마음대로 환각(Hallucination)을 일으키는 것을 완벽하게 통제합니다.Phase 3: Training-Free 다이나믹 멀티 샷 생성 (Attention Injection)이 논문의 기술적 하이라이트입니다. 사전 학습된 Diffusion 모델(예: AnimateDiff 등)의 가중치를 단 1도 건드리지 않고, Cross-Attention 계층의 연산을 추론(Inference) 단계에서 조작하여 일관성을 강제합니다.각 샷 $t$를 생성하는 노이즈 예측 과정 $\epsilon_\theta$에서, 텍스트 프롬프트 $p'_t$의 특징 벡터 외에도 1단계에서 추출한 **글로벌 앵커 $\mathcal{A}$의 텍스트/비주얼 임베딩을 Key($K$)와 Value($V$)에 강제로 주입(Inject)**합니다.표준 Cross-Attention 수식이 $\text{Attention}(Q, K, V) = \text{Softmax}\left(\frac{QK^T}{\sqrt{d}}\right)V$ 라면, 제안하는 Anchor-Injected Attention은 다음과 같습니다.$$\tilde{K} = [K_{p'_t} \parallel K_{\mathcal{A}}]$$$$\tilde{V} = [V_{p'_t} \parallel V_{\mathcal{A}}]$$$$\text{Attention}_{N2V} = \text{Softmax}\left(\frac{Q \tilde{K}^T}{\sqrt{d}}\right)\tilde{V}$$기존 프롬프트의 특징 행렬($K_{p'_t}, V_{p'_t}$)에 글로벌 앵커의 특징 행렬($K_{\mathcal{A}}, V_{\mathcal{A}}$)을 결합($\parallel$)함으로써, 모델은 매 프레임을 그릴 때마다 무조건 $\mathcal{A}$(예: 붉은 망토, 흉터)를 참조하게 됩니다. 재학습(Fine-tuning) 없이도 일관성이 유지되는 마법이 여기서 일어납니다.이 알고리즘의 핵심 기여도 (Contributions)진정한 N2V 달성: 사용자가 번거롭게 샷마다 프롬프트를 짤 필요 없이, 소설 텍스트 덩어리만 넣으면 알아서 시각적 요소를 추론하고 분리합니다.완전한 Training-Free: 로라(LoRA) 학습이나 추가적인 어댑터(Adapter) 훈련 없이, 오직 LLM의 디코딩과 Diffusion Attention Map 조작만으로 모달리티 갭을 해결합니다.SCENE (서사적/시각적 일관성) 붕괴 방지: 샷이 여러 번 넘어가도 초기 앵커($\mathcal{A}$)가 계속 Attention 연산에 개입하므로 엔티티가 증발하거나 변형되는 현상을 원천 차단합니다.
